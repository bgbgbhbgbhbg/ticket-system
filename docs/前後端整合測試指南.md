## 🐛 常見問題排查

### 1. 前端顯示 "API Error: 404"

**原因**: 後端沒有啟動或路由不對  
**解決**:
```bash
# 檢查後端是否運行
ps aux | grep dotnet

# 或直接訪問 Scalar UI 看是否可用
open http://localhost:5000/scalar/v1
```

### 2. 前端顯示 "API Error: CORS"

**原因**: CORS 策略未正確配置  
**檢查**:
- ✅ Program.cs 有 `app.UseCors("Frontend")`
- ✅ FrontendUrl 配置正確 (預設 http://localhost:3000)
- ✅ 中間件順序: 認證 → 授權 → CORS

### 3. 前端顯示 "API Error: 401/403"

**原因**: JWT 認證失敗  
**檢查**:
- ✅ Jwt:SecretKey 是否已設定: `dotnet user-secrets list`
- ✅ WeatherForecast 端點是否需要 `[Authorize]` 標記

### 4. 載入一直旋轉

**原因**: 網路請求超時  
**檢查**:
- ✅ 後端是否真的啟動了
- ✅ `NEXT_PUBLIC_API_URL` 是否設定正確
- ✅ 瀏覽器控制台 (F12) 有無具體錯誤資訊
